data-ride="carousel" 属性用于标记轮播在页面加载时就开始动画播放，无需使用初始化的js函数。使用 data 属性可以很容易控制轮播（Carousel）的位置。
属性 data-slide 接受关键字 prev 或 next，用来改变幻灯片相对于当前位置的位置。使用 data-slide-to 来向轮播床底一个原始滑动索引， 将把滑块移动到一个特定的索引，索引从 0 开始计数。
